# piano

这是一个钢琴软件，功能比较简单，以后会持续改进。

## 功能说明

1. **鼠标弹奏**：鼠标可以点击任何琴键，电脑会发出相应音符的声音。

2. **键盘弹奏（推荐）**：键盘可以点击 1（哆）到下一个八度的升 2（第二个八度的三和四目前只有鼠标能点击，因为键盘数量不够），一次可以点击多个按键，模拟真实钢琴弹奏。

3. **乐器切换**：琴键上三个按钮可以调节乐器种类（钢琴、笛子、弦乐）。

4. **视觉效果**：
   - 背景为星空，在不断移动（类似你在向前移动）
   - 弹奏时弹几个音符屏幕上会跳出来几个蓝色光点
   - 弹奏哪个琴键哪个按键就会亮一下，能显示你在弹哪个音符
   - 弹得快了星空顶部会有流星划过
